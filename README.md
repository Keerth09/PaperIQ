PaperIQ - Research Paper Analyzer

PaperIQ is a Streamlit based web application designed to analyze research papers efficiently. It extracts text from uploaded documents and provides structured summaries, issue detection, and useful visualization metrics. The goal of this project is to simplify research paper understanding and evaluation.

Project Description

This application allows users to upload research papers in PDF or DOCX format. The system processes the content, analyzes text quality, and generates meaningful insights including section summaries and issue identification. It also provides visual representations to help users better understand the document structure and performance.

Key Features

Extracts text from PDF and DOCX files
Generates structured section summaries
Detects writing or structural issues
Provides visualization dashboard for analysis
User friendly Streamlit interface

How to Run the Project

Step 1: Clone the repository
git clone https://github.com/Keerth09/PaperIQ.git

Step 2: Navigate to the project folder
cd PaperIQ

Step 3: Install required dependencies
pip install -r requirements.txt

Step 4: Run the application
streamlit run main.py
